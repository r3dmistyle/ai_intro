## Домашние задания

В этом разделе собраны домашние задания, по которым можно посмотреть примеры моего кода.

### 🟥 Средняя сложность

- **hw** — задания, в которых обычно требуется написать код по заданному условию.
- **case** — задания, где нужно с нуля предложить и реализовать собственное решение поставленной проблемы.

### ⬛ Повышенная сложность

- **hw** — задания, в которых обычно требуется написать код по заданному условию.
- **kaggle** — задания, где нужно с нуля решить поставленную задачу, включая анализ данных, выбор подхода и реализацию решения.
